# Cheese Thief Sound Effects

This directory contains sound effects for the Cheese Thief game.

## Required Sound Files

Download the following sound effects from Pixabay (https://pixabay.com/sound-effects/search/rat/ or similar):

### Core Sounds
1. **die-roll.mp3** - Die rolling sound for the rolling phase
2. **mouse-squeak-1.mp3** - Mouse squeak variation 1
3. **mouse-squeak-2.mp3** - Mouse squeak variation 2
4. **mouse-squeak-3.mp3** - Mouse squeak variation 3
5. **sneaking.mp3** - Subtle sneaking/scurrying sound (can be looped)
6. **peek.mp3** - Quick peek/spy sound effect
7. **cheese-munch.mp3** - Eating/munching sound (cheese theft)
8. **whisper.mp3** - Whispering/scheming sound for accomplice selection
9. **voting-chatter.mp3** - Mouse chatter/discussion (can be looped)
10. **victory.mp3** - Victorious/celebratory mouse sound
11. **defeat.mp3** - Defeated/sad sound

## Sound Requirements

- **Format**: MP3 (best browser compatibility)
- **Size**: Keep under 100KB per file for fast loading
- **Length**: Short clips (0.5-3 seconds) for best responsiveness
- **Volume**: Normalize all files to similar volume levels before adding

## Recommended Sources

- Pixabay Sound Effects: https://pixabay.com/sound-effects/search/rat/
- Pixabay Sound Effects: https://pixabay.com/sound-effects/search/mouse/
- Freesound.org (requires attribution for some sounds)
- ZapSplat (free with account)

## Installation

1. Download appropriate sound effects from the sources above
2. Convert to MP3 format if necessary
3. Normalize volume levels using audio editing software (Audacity is free)
4. Rename files to match the names listed above
5. Place all files in this directory

## Testing

After adding sound files, test them in the game by:
1. Running `npm run dev` or `composer run dev`
2. Playing through a complete game round
3. Verifying each sound plays at the appropriate time
4. Checking that volume levels are balanced
